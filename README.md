# 概要

前回の会議から時間が空いてしまうと、議題内容や課されていた宿題を忘れてしまう場合がある。

それを防ぐため、生成AIを活用し、議事録の要約や、課されていた宿題(者)を教えてくれる業務改善アプリを作成。
